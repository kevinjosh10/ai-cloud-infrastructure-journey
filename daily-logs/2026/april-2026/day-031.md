### 2️⃣ Understood IPv6 Address Structure
- IPv6 consists of 8 groups (hextets)
- Each group contains 4 hexadecimal digits
- Uses base-16 (0–9, a–f)

Example:
    2001:0db8:85a3:0000:0000:8a2e:0370:7334

---

### 3️⃣ Learned IPv6 Abbreviation Rules

#### Removed Leading Zeros
    0db8 → db8  
    0370 → 370  

#### Compressed Consecutive Zeros
    2001:0db8:85a3:0000:0000:8a2e:0370:7334  
    ↓  
    2001:db8:85a3::8a2e:370:7334  

Rules:
- `::` can be used only once
- Never use `:::`
- Cannot remove blocks completely

---

### 4️⃣ Learned Key IPv6 Address Types

#### Global Unicast
- Range: 2000::/3  
- Public internet communication  
- Equivalent to public IPv4  

#### Link-Local
- Range: fe80::/10  
- Automatically assigned  
- Used within same network  

#### Loopback
    ::1  

- Refers to local machine  
- Equivalent to 127.0.0.1  

---

### 5️⃣ Understood IPv6 in Cloud

- Cloud platforms support dual-stack networking (IPv4 + IPv6)
- Important for scalability and future infrastructure
- Used in VPCs, load balancers, and distributed systems
