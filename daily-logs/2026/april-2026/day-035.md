### 1️⃣ Practiced Subnetting
- Solved multiple subnetting problems using CIDR notation
- Reviewed common subnet sizes and usable host counts

Example:
    Network: 192.168.1.0/24  
    Broadcast: 192.168.1.255  
    Usable Range: 192.168.1.1 – 192.168.1.254  

Memorized:
    /30 → 4 IPs (2 usable)  
    /28 → 16 IPs (14 usable)  
    /24 → 256 IPs (254 usable)

### 2️⃣ Performed DNS Queries
- Used `dig` to query different DNS record types
- Explored DNS resolution path using `+trace`

Commands used:
    dig google.com  
    dig google.com MX  
    dig google.com TXT  
    dig google.com CNAME  
    dig +trace google.com  

Observed:
- A records return IPv4 addresses  
- MX records show mail servers  
- TXT records used for verification  
- +trace shows full DNS resolution path

Understood:
