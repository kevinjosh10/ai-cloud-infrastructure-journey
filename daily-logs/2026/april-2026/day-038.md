### 3️⃣ Understood TCP vs UDP

#### TCP (Transmission Control Protocol)
- Connection-oriented
- Reliable and ensures data delivery
- Used in critical applications

Examples:
- SSH (22)
- HTTP (80)
- HTTPS (443)
- Databases

#### UDP (User Datagram Protocol)
- Connectionless
- Faster but no delivery guarantee
- Used where speed is priority

Examples:
- DNS (default)
- Streaming services

---

### 4️⃣ Learned DNS Uses Both TCP & UDP
- UDP → Used for normal queries (fast)
- TCP → Used for large responses (reliable)

### 6️⃣ Checked Listening Ports

Command used:
    ss -tulpn

Understood:
- `t` → TCP  
- `u` → UDP  
- `l` → Listening  
- `p` → Process  

Example:
    tcp LISTEN 0 128 0.0.0.0:80  

Confirmed:
- Port 80 is open
- A service (like nginx) is listening

---

### 7️⃣ Understood Why Web Servers Use Port 80
- Web servers like nginx listen on port 80 by default
- Handles incoming HTTP requests from browsers

Flow:
    Browser → Port 80 → Web Server → Response

Example:
