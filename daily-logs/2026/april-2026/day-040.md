### 2️⃣ Explored Load Balancing Algorithms

#### Round Robin
- Requests distributed sequentially across servers  
- Simple but does not consider server load  

Example:
    Req1 → Server A  
    Req2 → Server B  
    Req3 → Server C  
    Req4 → Server A  

#### Least Connections
- Sends traffic to server with least active connections  
- More efficient and widely used  

#### IP Hash
- Uses client IP to determine server  
- Same user always connects to same server  

#### Weighted Round Robin
- Traffic distributed based on server capacity  

Example:
    Server A → 70%  
    Server B → 30%  

---

### 3️⃣ Learned Load Balancer Types

#### Layer 4 (Transport Layer)
- Works at TCP/UDP level  
- Uses IP address and port  
- Faster but cannot inspect content  

#### Layer 7 (Application Layer)
- Works at HTTP/HTTPS level  
- Can inspect request content  
- Supports intelligent routing  

Example:
    /api → Server A  
    /images → Server B  

---

### 4️⃣ Mapped Load Balancers to AWS

#### Application Load Balancer (ALB)
- Works at Layer 7  
- Supports HTTP/HTTPS  
- Enables path-based routing  
- Used in microservices  

#### Network Load Balancer (NLB)
- Works at Layer 4  
- Supports TCP/UDP  
- High performance and low latency  

---

### 5️⃣ Understood VPN Fundamentals

- VPN creates a secure encrypted tunnel over the internet  
- Protects data from interception  
- Used for secure communication between users and networks  

Without VPN:
    User → Internet → Server (Not Secure)  

With VPN:
    User → Encrypted Tunnel → Server (Secure)  

---

### 6️⃣ Learned VPN Types

#### Site-to-Site VPN
- Connects two networks  
- Example: Office Network ↔ AWS VPC  
- Used for permanent connections  

#### Client VPN
- Connects individual user to network  
- Example: Laptop → Company Network  
- Used for remote access  

---

### 7️⃣ Explored AWS Connectivity Services

#### AWS VPN Gateway
- Connects on-premise network to AWS VPC  
- Used in Site-to-Site VPN  

#### AWS Direct Connect
- Private connection to AWS (not over internet)  
- Provides low latency and high bandwidth  
- More secure but expensive  

Example:
    Data Center → Direct Connect → AWS
