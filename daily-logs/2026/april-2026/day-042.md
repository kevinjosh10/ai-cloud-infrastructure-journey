### 3️⃣ Revisited DNS & DHCP Concepts
- DNS:
    Converts domain names to IP addresses  

- DHCP:
    Automatically assigns IP addresses to devices  

Real-world scenarios:
- DNS failure → Website not loading  
- DHCP failure → No internet connectivity  

---

### 4️⃣ Strengthened HTTP/HTTPS & TLS Knowledge
- HTTP:
    Unencrypted communication  

- HTTPS:
    Encrypted using SSL/TLS  

#### TLS Handshake:
1. Client Hello  
2. Server Hello  
3. Certificate Exchange  
4. Key Exchange  
5. Secure Connection Established  

- Understood how secure communication is built  

---

### 5️⃣ Reviewed Ports, Firewalls & Load Balancers
- Common ports:
    80 → HTTP  
    443 → HTTPS  
    22 → SSH  

- Firewalls / Security Groups:
    Control inbound and outbound traffic  

- Load Balancer:
    Distributes traffic across multiple servers  
    Improves availability and scalability
