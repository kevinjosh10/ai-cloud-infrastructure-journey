## ✅ What I Completed

### 1️⃣ Set Up Python Environment
- Verified Python installation using:
    python3 --version  
- Encountered missing `venv` issue and fixed it by installing:
    sudo apt install python3-venv  
- Created virtual environment:
    python3 -m venv cloudenv  
- Activated virtual environment:
    source cloudenv/bin/activate  

Confirmed:
- Virtual environment successfully created  
- `(cloudenv)` visible in terminal
