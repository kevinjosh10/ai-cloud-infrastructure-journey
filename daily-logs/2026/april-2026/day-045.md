## ✅ What I Completed

### 1️⃣ Learned Lists Fundamentals
- Understood lists are ordered, mutable collections  
- Learned indexing starts from 0  
- Practiced creating lists for storing multiple values  

Example:
    servers = ['web-01', 'web-02', 'db-01']

Access:
    servers[0]   → web-01  
    servers[-1]  → db-01  

---

### 2️⃣ Practiced List Methods

- `.append()` → add element  
- `.remove()` → remove specific element  
- `.sort()` → sort list  
- `.reverse()` → reverse order  
- `.pop()` → remove last element  
- `len()` → get total elements  

Example:
    servers.append('web-03')  
    servers.remove('db-01')  
    servers.sort()  
    servers.reverse()  
    servers.pop()  
    len(servers)  

---

### 3️⃣ Learned List Slicing

- Extracted parts of lists using slicing  
- Used for accessing subsets of data  

Examples:
    servers[0:2] → first 2 elements  
    servers[:3]  → first 3 elements  
    servers[-1]  → last element  

---

### 4️⃣ Learned List Comprehensions

- Created new lists in a single line  
- Improved code readability and efficiency  

Examples:
    [s.upper() for s in servers]  

Filtering:
    [s for s in servers if 'web' in s]  

---

### 5️⃣ Learned Tuples (Immutable Data)

- Understood tuples are ordered but immutable  
- Cannot modify after creation  
- Used for fixed configurations  

Example:
    config = ('prod', '10.0.0.1', 443)

Access:
    config[0] → prod  

---

### 6️⃣ Completed Practice Task

Created AWS regions list:
    regions = ['us-east-1', 'us-west-2', 'ap-south-1', 'eu-west-1', 'us-east-2']

Sorted regions:
    regions.sort()

Filtered only US regions:
    us_regions = [r for r in regions if r.startswith('us-')]
