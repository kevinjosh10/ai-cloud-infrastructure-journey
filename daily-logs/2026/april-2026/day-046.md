## ✅ What I Completed

### 1️⃣ Learned Dictionaries (Key-Value Pairs)
- Understood dictionaries store data as key-value pairs  
- Similar to JSON used in cloud APIs and AWS responses  
- Keys act as identifiers, values store actual data  

Example:
    instance = {'id': 'i-abc123', 'type': 't2.micro', 'state': 'running'}

---

### 2️⃣ Learned Dictionary Operations
- Accessed values using keys  
- Used `.get()` for safe access with default values  
- Retrieved all keys, values, and key-value pairs  

Examples:
    instance['id']  
    instance.get('key', 'default')  

    instance.keys()  
    instance.values()  
    instance.items()  

---

### 3️⃣ Learned Dictionary Comprehensions
- Used to transform dictionary data efficiently  
- Helps in automation and data processing  

Example:
    tags = {'env': 'dev', 'owner': 'kevin'}  

    upper_tags = {k: v.upper() for k, v in tags.items()}  

---

### 4️⃣ Learned Sets (Unique & Unordered)
- Sets store only unique values  
- Automatically remove duplicates  
- Useful for comparisons and filtering  

Example:
    numbers = {1, 2, 3}  

---

### 5️⃣ Practiced Set Operations
- Performed union, intersection, and difference  

Examples:
    a = {'us-east-1', 'eu-west-1'}  
    b = {'us-east-1', 'ap-south-1'}  

    a | b  
    a & b  
    a - b  

---

### 6️⃣ Modeled AWS EC2 Instance (Hands-On)
- Created a dictionary representing a real EC2 instance  
- Included nested dictionary for tags  

Example:
    instance = {  
        'instance_id': 'i-123456',  
        'type': 't2.micro',  
        'state': 'running',  
        'region': 'ap-south-1',  
        'tags': {  
            'env': 'dev',  
            'owner': 'kevin'  
        }  
    }  

---

### 7️⃣ Wrote Functions to Get/Set Values

Get value:
    def get_instance_value(inst, key):  
        return inst.get(key, 'Not Found')  

Set value:
    def set_instance_value(inst, key, value):  
        inst[key] = value  

Get tag:
    def get_tag(inst, tag_key):  
        return inst['tags'].get(tag_key, 'No Tag')  

Set tag:
    def set_tag(inst, tag_key, tag_value):  
        inst['tags'][tag_key] = tag_value
