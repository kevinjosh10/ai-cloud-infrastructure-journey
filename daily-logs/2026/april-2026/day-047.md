### 2️⃣ Used `enumerate()` for Index Tracking

- Learned how to access both index and value  
- Useful for logging and debugging in scripts  

Example:
    for i, server in enumerate(servers):

### 3️⃣ Iterated Through Dictionaries

- Learned to loop through key-value pairs  
- Important for handling structured cloud data (JSON-like)  

Example:
    instance = {"id": "i-123", "state": "running"}

    for key, value in instance.items():
        print(key, ":", value)

---

### 4️⃣ Learned `while` Loops (Retry Logic)

- Used when number of iterations is unknown  
- Implemented retry mechanism logic  

Example:
    retries = 0

    while retries < 3:
        print("Retrying...")
        retries += 1

---

### 5️⃣ Learned Conditionals (`if / elif / else`)

- Used conditional logic to make decisions  
- Applied filtering based on state  

Example:
    state = "running"

    if state == "running":
        print("Active")
    elif state == "stopped":
        print("Stopped")
    else:
        print("Unknown")

---

### 6️⃣ Learned Loop Control Statements

#### `break`
- Exits loop immediately  

Example:
    for i in range(5):
        if i == 3:
            break
        print(i)

#### `continue`
- Skips current iteration  

Example:
    for i in range(5):
        if i == 2:
            continue
        print(i)

#### `pass`
- Placeholder for future logic  

Example:
    for i in range(5):
        pass

---

### 7️⃣ Built Practice Script (Cloud Use Case)

- Created a script to filter EC2-like instances  
- Printed only instances in `running` state and `us-east-1` region  

Example:
    instances = [
        {"id": "i-001", "state": "running", "region": "us-east-1"},
        {"id": "i-002", "state": "stopped", "region": "us-east-1"},
        {"id": "i-003", "state": "running", "region": "us-west-2"},
        {"id": "i-004", "state": "running", "region": "us-east-1"},
    ]

    for instance in instances:
        if instance["state"] == "running" and instance["region"] == "us-east-1":
            print("Instance ID:", instance["id"])
