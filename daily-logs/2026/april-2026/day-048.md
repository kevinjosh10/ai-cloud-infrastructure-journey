### 3️⃣ Learned *args (Flexible Positional Arguments)
- Learned how to pass multiple arguments dynamically
- Useful when number of inputs is unknown

Example:
    def print_services(*services):
        for service in services:
            print(service)

    print_services("EC2", "S3", "Lambda")

---

### 4️⃣ Learned **kwargs (Flexible Keyword Arguments)
- Learned how to pass dynamic key-value arguments
- Useful for handling configurations (like AWS API inputs)

Example:
    def create_instance(**details):
        for key, value in details.items():
            print(f"{key}: {value}")

    create_instance(instance_type="t2.micro", region="ap-south-1")

### 6️⃣ Built a Custom Module (Hands-On)

Created:
    aws_helpers.py

Functions implemented:

    import datetime

    def format_instance_id(instance_id):
        return instance_id.strip().lower()

    def get_region_from_az(availability_zone):
        return availability_zone[:-1]

    def calculate_runtime_hours(start_time):
        now = datetime.datetime.now()
        runtime = now - start_time
        return round(runtime.total_seconds() / 3600, 2)

### 7️⃣ Used Module in Main Script

Created:

Example:

    print("Instance ID:", formatted_id)
    print("Region:", region)
    print("Runtime (hours):", runtime)
