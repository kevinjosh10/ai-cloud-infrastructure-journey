---

### 2️⃣ Learned File Writing

#### Write Mode (`w`)
- Overwrites existing file content  
- Used for fresh outputs  

#### Append Mode (`a`)
- Adds data without deleting existing content  
- Useful for logging and tracking  

Understood when to use each mode in real-world scenarios  

---

### 3️⃣ Learned JSON Fundamentals

- JSON = JavaScript Object Notation  
- Standard format for data exchange in cloud systems  
- Similar to Python dictionaries and lists  

Understood:
- Key-value structure  
- Arrays (lists of objects)  
- Nested data formats  

---

### 4️⃣ Learned JSON Operations

#### Parsing (JSON → Python)
- Convert JSON string into usable data  

#### Serialization (Python → JSON)
- Convert Python data into JSON format  

Key Tools:
- `json.loads()` → parse string  
- `json.dumps()` → convert to JSON  

---

### 5️⃣ Learned JSON File Handling

#### Reading JSON File
- Open file  
- Load JSON data  
- Use it in program  

#### Writing JSON File
- Prepare processed data  
- Save into new JSON file  
- Use indentation for readability
