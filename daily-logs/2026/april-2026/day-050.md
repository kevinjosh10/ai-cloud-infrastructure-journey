- Understood how to handle runtime errors using try/except

Example:
    try:
        x = int("abc")

Example:
    try:

### 3️⃣ Learned Common Python Exceptions

#### ValueError
- Occurs when invalid value is used  
    int("abc")

#### TypeError
- Occurs when wrong data types are used together  
    "2" + 2  

#### KeyError
- Occurs when accessing a missing dictionary key  
    data["missing"]

#### FileNotFoundError
- Occurs when file does not exist  
    open("config.json")

#### ConnectionError
- Occurs during network/API failures  

---

### 4️⃣ Learned Raising Custom Exceptions
- Used `raise` to enforce rules in code
- Added context to error messages for better debugging

Example:
    def get_region(region):
        valid_regions = ["us-east-1", "ap-south-1"]

        if region not in valid_regions:
            raise ValueError(f"Invalid region: {region}")

        return region

### 5️⃣ Understood Exception Hierarchy
- Learned all exceptions inherit from BaseException
- Understood importance of catching specific errors before general ones

Structure:
    BaseException
        └── Exception
            ├── ValueError
            ├── TypeError
            ├── KeyError
            └── FileNotFoundError

Example:
    try:
        pass
    except ValueError:
        print("Value issue")
    except Exception:
        print("General error")

### 6️⃣ Built a Production-Ready JSON Loader
- Handled file not found errors
- Handled invalid JSON format
- Validated required keys
- Logged descriptive error messages
- Prevented crashes using fallback return

Example:
    import json

    def load_config(file_path):
        try:
            with open(file_path, "r") as f:
                data = json.load(f)

            if "region" not in data:
                raise KeyError("Missing 'region' in config")

            return data

        except FileNotFoundError:
            print(f"[ERROR] File not found: {file_path}")

        except json.JSONDecodeError:
            print(f"[ERROR] Invalid JSON format in file: {file_path}")

        except KeyError as e:
            print(f"[ERROR] {e}")

        except Exception as e:
            print(f"[ERROR] Unexpected error: {e}")

        finally:
            print("[INFO] Config load attempt finished")

        return None
