### 2️⃣ Performed GET Requests

- Installed and used `requests` library  
- Sent GET requests to fetch data from APIs  
- Checked response status codes  

Example:
    import requests

    response = requests.get("https://api.github.com")
    print(response.status_code)
    print(response.json())

---

### 3️⃣ Learned POST Requests

- Sent data to server using POST method  
- Used JSON payloads in requests  

Example:
    import requests

    data = {"name": "Kevin", "role": "AI Engineer"}
    response = requests.post("https://httpbin.org/post", json=data)

    print(response.json())

---

### 4️⃣ Understood Headers & Authentication

- Learned how headers are used for API communication  
- Used Authorization tokens in requests  
- Understood role of headers in security and data formatting  

Example:
    headers = {
        "Authorization": "Bearer YOUR_TOKEN"
    }

---

### 5️⃣ Learned Session Management

- Used `requests.Session()` to reuse connections  
- Improved efficiency and performance  
- Stored headers across multiple requests  

Example:
    session = requests.Session()
    session.headers.update({"Authorization": "Bearer TOKEN"})

    session.get("https://api.example.com/data")

---

### 6️⃣ Implemented Timeout Handling

- Learned importance of timeouts in production  
- Prevented application hanging  

Example:
    import requests

    try:
        response = requests.get("https://api.github.com", timeout=5)
        print(response.json())
    except requests.exceptions.Timeout:
        print("Request timed out")

---

### 7️⃣ Completed Hands-On Practice (GitHub API)

- Fetched repositories using GitHub API  
- Parsed JSON response  
- Extracted repository name and language  

Example:
    import requests

    username = "kevinjosh10"
    url = f"https://api.github.com/users/{username}/repos"

    response = requests.get(url)

    for repo in response.json():
        print(repo["name"], "-", repo["language"])
