### 2️⃣ Understood boto3 Interfaces

#### Client (Low-Level)
- Direct access to AWS APIs  
- More control and detailed responses  
- Used for production-level scripting  

#### Resource (High-Level)
- Object-oriented interface  
- Easier to use and more Pythonic  
- Good for quick development  

---

### 3️⃣ Configured AWS Credentials

Command used:

Configured:

Confirmed:
