### 2️⃣ Retrieved CloudWatch Metrics (EC2 Monitoring)
- Used `cloudwatch.get_metric_statistics()`  
- Retrieved CPU utilization of EC2 instance  
- Learned how monitoring works in AWS  

Example:
    from datetime import datetime, timedelta

    cw = boto3.client('cloudwatch')

    end = datetime.utcnow()
    start = end - timedelta(hours=1)

    response = cw.get_metric_statistics(
        Namespace='AWS/EC2',
        MetricName='CPUUtilization',
        Dimensions=[
            {'Name': 'InstanceId', 'Value': 'YOUR_INSTANCE_ID'}
        ],
        StartTime=start,
        EndTime=end,
        Period=300,
        Statistics=['Average']
    )

### 4️⃣ Learned Pagination in boto3
- Understood AWS limits response size  
- Learned to use paginators for large datasets  
- Implemented `get_paginator()`  

Example:
    s3 = boto3.client('s3')

    paginator = s3.get_paginator('list_objects_v2')

    for page in paginator.paginate(Bucket='my-bucket'):
        for obj in page.get('Contents', []):
            print(obj['Key'])

### 5️⃣ Built Cloud Security Audit Script
- Created `cloud_audit.py`  
- Identified IAM users without MFA enabled  
- Used `list_users()` and `list_mfa_devices()`  
- Applied pagination for reliability  

Example:
    import boto3

    iam = boto3.client('iam')

    paginator = iam.get_paginator('list_users')

    for page in paginator.paginate():
        for user in page['Users']:
            username = user['UserName']
            mfa = iam.list_mfa_devices(UserName=username)

            if len(mfa['MFADevices']) == 0:
                print(f"{username} has NO MFA enabled")
