### 4️⃣ Learned Static & Class Methods

#### Static Method
- Does not use `self`  
- Used for utility/helper functions  

Example:
    @staticmethod
    def validate_region(region):
        return region.startswith("ap-")  

#### Class Method
- Uses `cls` instead of `self`  
- Used as alternative constructors  

Example:
    @classmethod
    def default_region(cls):
        return cls("ap-south-1")  

---

### 5️⃣ Learned `__str__` and `__repr__`
- Improved object readability and debugging  
- `__str__` → user-friendly output  
- `__repr__` → developer/debug output  

Example:
    def __str__(self):
        return f"EC2Manager in {self.region}"

    def __repr__(self):
        return f"EC2Manager(region='{self.region}')"

### 6️⃣ Built Real Cloud Resource Classes
- Created a base class `CloudResource`

- Implemented subclasses for EC2 and S3  

#### Base Class:
    class CloudResource:
        def __init__(self, name, region):
            self.name = name
            self.region = region

        def create(self):
            raise NotImplementedError

        def delete(self):
            raise NotImplementedError

        def status(self):
            raise NotImplementedError  

#### EC2 Subclass:
    class EC2Instance(CloudResource):
        def create(self):
            print(f"Creating EC2 instance: {self.name}")  

#### S3 Subclass:
    class S3Bucket(CloudResource):
        def create(self):
            print(f"Creating S3 bucket: {self.name}")
