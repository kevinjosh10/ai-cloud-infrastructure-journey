---

### 6️⃣ Practiced Testing Without AWS Calls
- Mocked boto3 responses
- Simulated AWS EC2 data
- Verified logic without hitting real infrastructure

Example:
    @patch("boto3.client")
    def test_ec2(mock_boto):
        mock_ec2 = mock_boto.return_value
        mock_ec2.describe_instances.return_value = {"Reservations": []}

        result = mock_ec2.describe_instances()

        assert "Reservations" in result

---

### 7️⃣ Tested Real Project Logic
- Wrote tests for EC2 parsing functions

Example:
    def extract_instance_ids(data):
        ids = []
        for r in data["Reservations"]:
            for i in r["Instances"]:
                ids.append(i["InstanceId"])
        return ids

    def test_extract_instance_ids():
        mock_data = {
            "Reservations": [
                {
                    "Instances": [
                        {"InstanceId": "i-123"},
                        {"InstanceId": "i-456"}
                    ]
                }
            ]
        }

        assert extract_instance_ids(mock_data) == ["i-123", "i-456"]
