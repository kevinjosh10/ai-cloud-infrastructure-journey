### 2️⃣ Understood Positional vs Optional Arguments

#### Positional Arguments
- Required inputs without `--`
- Example:
    parser.add_argument("region")

Usage:
    python script.py us-east-1

#### Optional Arguments
- Defined using `--`
- Can have default values

Example:
    parser.add_argument("--profile", default="default")

Usage:
    python script.py us-east-1 --profile dev

### 3️⃣ Learned Argument Types & Validation
- Used `type=int`, `type=str` for input validation
- Restricted inputs using `choices`
- Enforced required arguments using `required=True`
- Added help descriptions for clarity

Example:
    parser.add_argument("--action", choices=["start", "stop"])
    parser.add_argument("--state", required=True, help="Instance state filter")
