---

## 👤 User Management

### 🔹 Create a new user
```bash
sudo useradd -m newuser
```

### 🔹 Set password
```bash
sudo passwd newuser
```

### 🔹 Delete user
```bash
sudo userdel -r newuser
```

### 🔹 Key Concepts
- Each user has a **UID (User ID)**
- Default home directory: `/home/username`
- Default shell: `/bin/bash`

---

## 👥 Group Management

### 🔹 Create a group
```bash
sudo groupadd devteam
```

### 🔹 Add user to group
```bash
sudo usermod -aG devteam kevin
```

### 🔹 Check user groups
```bash
groups kevin
```

### 🔹 Key Insight
- Groups simplify permission management
- Instead of assigning permissions individually, assign them to groups

---

## 📂 Important System Files

### 📄 `/etc/passwd`
```bash
cat /etc/passwd
```
- Stores user account details (username, UID, home directory, shell)

---

### 🔒 `/etc/shadow`
```bash
sudo cat /etc/shadow
```
- Stores **encrypted passwords**
- Accessible only by root

---

### 👥 `/etc/group`
```bash
cat /etc/group
```
- Stores group information and members

## ⚡ Sudo & Privilege Management

### 🔹 What is sudo?
- `sudo` = **SuperUser DO**
- Allows normal users to execute commands with elevated privileges

---

### 🔹 Example
```bash
sudo apt update
```

---

### 🔹 Edit sudo permissions safely
```bash
sudo visudo
```

---

### 🔹 Example sudoers entry
```bash
kevin ALL=(ALL) ALL
```
- Full administrative access

---

### 🔥 Advanced Control
```bash
kevin ALL=(ALL) /usr/bin/systemctl restart nginx
```
- Allows only specific command execution

---

## 🔄 Switching Users

```bash
su - newuser
```

- Switches to another user account
- Loads their environment

## 🧪 Hands-on Practice (EC2)

### ✅ Step 1: Create user
```bash
sudo useradd -m devuser
sudo passwd devuser
```

### ✅ Step 2: Create group
```bash
sudo groupadd devteam
```

### ✅ Step 3: Add user to group
```bash
sudo usermod -aG devteam devuser
```

### ✅ Step 4: Switch user
```bash
su - devuser
```

### ✅ Step 5: Test permissions
```bash
sudo apt update
```
- Should fail (no sudo access)

---

### 🔥 Optional: Grant sudo access
```bash
sudo visudo
```

Add:
```bash
devuser ALL=(ALL) ALL
```
