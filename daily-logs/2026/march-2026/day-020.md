## ⏰ Cron Jobs (Task Automation)

### 🔹 What is Cron?
- Cron is a **time-based job scheduler in Linux**
- Used to automate repetitive tasks

---

### 🔹 Cron Syntax
    * * * * *
    | | | | |
    | | | | └── Day of week (0–7)
    | | | └──── Month (1–12)
    | | └────── Day of month (1–31)
    | └──────── Hour (0–23)
    └────────── Minute (0–59)

---

### 🔹 Common Examples
    * * * * *        → Every minute
    0 * * * *        → Every hour
    0 9 * * *        → Every day at 9 AM
    0 0 * * 0        → Every Sunday

---

## ⚙️ Creating a Cron Job

### 🔹 Step 1: Open crontab
    crontab -e

---

### 🔹 Step 2: Add a job
    * * * * * date >> /home/yourusername/cronlog.txt

---

### 🔹 Step 3: Verify output
    cat /home/yourusername/cronlog.txt

## ⚙️ systemd (Service Management)

### 🔹 What is systemd?
- systemd is a **service manager for Linux**
- Used to control background services (servers, scripts, daemons)

---

### 🔹 Basic Commands
    sudo systemctl start nginx
    sudo systemctl stop nginx
    sudo systemctl restart nginx

---

### 🔹 Enable/Disable at Boot
    sudo systemctl enable nginx
    sudo systemctl disable nginx

---

## 📊 Monitoring Services

### 🔹 Check service status
    systemctl status nginx

---

### 🔹 View logs
    journalctl -u nginx

---

### 🔹 Live logs
    journalctl -u nginx -f

---

## 🔧 Creating a Custom systemd Service

### 🔹 Step 1: Create Python Script
    nano /home/yourusername/hello.py

Paste:
    import time

    while True:
        with open("/home/yourusername/service_log.txt", "a") as f:
            f.write("Service is running\n")
        time.sleep(10)

---

### 🔹 Step 2: Create Service File
    sudo nano /etc/systemd/system/myservice.service

Paste:
    [Unit]
    Description=My Python Service
    After=network.target

    [Service]
    ExecStart=/usr/bin/python3 /home/yourusername/hello.py
    Restart=always
    User=yourusername

    [Install]
    WantedBy=multi-user.target

---

### 🔹 Step 3: Reload systemd
    sudo systemctl daemon-reexec
    sudo systemctl daemon-reload

---

### 🔹 Step 4: Start Service
    sudo systemctl start myservice

---

### 🔹 Enable at Boot
    sudo systemctl enable myservice

---

### 🔹 Step 5: View Logs
    journalctl -u myservice -f

## 🧪 Hands-on Practice (EC2)

### ✅ Step 3: Modify Python service
Change script to:

    from datetime import datetime

    while True:
