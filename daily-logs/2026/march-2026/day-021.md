## ⏰ Cron Job Automation (Revision + Implementation)

### 🔹 Script Created
    #!/bin/bash
    echo "Server time: $(date)" >> ~/server.log

---

### 🔹 Making executable
    chmod +x logtime.sh

---

### 🔹 Scheduling with cron
    crontab -e

Add:
    */5 * * * * /home/kevz/logtime.sh

---

### 🔹 Verification
    cat ~/server.log

---

### ⚡ Outcome
- Automated logging every 5 minutes
- Confirmed cron execution
- Built a basic monitoring mechanism

## 👤 Users, Groups & Sudo (Access Control)

### 🔹 Creating user
    sudo useradd -m devuser
    sudo passwd devuser

---

### 🔹 Granting sudo access
    sudo usermod -aG sudo devuser

---

### 🔹 Switching user
    su - devuser

---

### 🔹 Testing privileges
    sudo apt update

---

### 🔐 Outcome
- Learned how Linux controls access
- Understood privilege escalation using sudo
- Simulated real multi-user system management
