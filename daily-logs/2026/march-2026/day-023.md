## 🔀 If / Elif / Else Statements

### 🔹 Basic syntax
    if [ condition ]; then
        # execute if true
    elif [ condition ]; then
        # execute if another condition is true
    else
        # fallback case
    fi

---

### 🔹 Example
    #!/bin/bash
    NUM=10

    if [ $NUM -gt 5 ]; then
        echo "Number is greater than 5"
    else
        echo "Number is 5 or less"
    fi

## 🔢 Numeric Comparisons

### 🔹 Operators
- `-eq` → equal  
- `-ne` → not equal  
- `-gt` → greater than  
- `-lt` → less than  
- `-ge` → greater than or equal  
- `-le` → less than or equal  

---

### 🔹 Example
    #!/bin/bash
    A=20
    B=15

    if [ $A -gt $B ]; then
        echo "A is greater than B"
    fi

---

### ⚡ Outcome
- Compared numerical values in scripts  
- Built logic based on system metrics  
- Enabled decision-driven automation

## 🔤 String Comparisons

### 🔹 Operators
- `==` → equal  
- `!=` → not equal  
- `-z` → empty string  
- `-n` → non-empty string  

---

### 🔹 Example
    #!/bin/bash
    NAME="Kevin"

    if [ "$NAME" == "Kevin" ]; then
        echo "Welcome Kevin"
    fi

### ⚡ Outcome

## 📁 File Condition Checks

### 🔹 Operators
- `-f` → file exists  
- `-d` → directory exists  
- `-r` → readable  
- `-x` → executable  

---

### 🔹 Example
    #!/bin/bash
    if [ -f "test.txt" ]; then
        echo "File exists"
    else
        echo "File does not exist"
    fi

### ⚡ Outcome

## 🧪 Practice Script — File Checker

### 🔹 Script Created
    #!/bin/bash

    FILE="myfile.txt"

    if [ -f "$FILE" ]; then
        echo "File exists"

        SIZE=$(stat -c%s "$FILE")
        echo "File size: $SIZE bytes"
    else
        echo "File not found. Creating file..."
        touch "$FILE"
        echo "File created"
    fi
