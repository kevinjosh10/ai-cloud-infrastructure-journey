## 🔁 For Loop (Iteration-Based Execution)

### 🔹 Basic syntax
    for i in list
    do
        # commands
    done

---

### 🔹 Example
    #!/bin/bash

    for i in 1 2 3
    do
        echo $i
    done

---

### 🔹 Using range
    #!/bin/bash

    for i in {1..5}
    do
        echo "Number: $i"
    done

---

### 🔹 Using seq
    #!/bin/bash

    for i in $(seq 1 10)
    do
        echo $i
    done

---

### ⚡ Outcome
- Learned iteration using loops  
- Automated repeated execution  
- Applied loops to sequences and ranges

## 🔄 While Loop (Condition-Based Execution)

### 🔹 Syntax
    while [ condition ]
    do
        # commands
    done

---

### 🔹 Example
    #!/bin/bash

    count=1

    while [ $count -le 5 ]
    do
        echo "Count: $count"
        count=$((count+1))
    done

---

### ⚡ Outcome
- Executed loops based on conditions  
- Learned dynamic looping (unknown iterations)  
- Built logic-driven repetition  

---

## 🔁 Until Loop (Reverse Condition Loop)

### 🔹 Syntax
    until [ condition ]
    do
        # commands
    done

---

### 🔹 Example
    #!/bin/bash

    count=1

    until [ $count -gt 5 ]
    do
        echo "Count: $count"
        count=$((count+1))
    done

---

### ⚡ Outcome
- Understood reverse looping logic  
- Learned difference between while vs until  
- Expanded control over loop execution  

---

## 🧠 Loop Control Statements

### 🔴 break (exit loop completely)
    for i in {1..10}
    do
        if [ $i -eq 5 ]; then
            break
        fi
        echo $i
    done

---

### 🟡 continue (skip current iteration)
    for i in {1..5}
    do
        if [ $i -eq 3 ]; then
            continue
        fi
        echo $i
    done

---

### ⚡ Outcome
- Controlled loop execution flow  
- Skipped unnecessary iterations  
- Built efficient scripts

## 🧪 Practice Script — Backup Automation (Script 1/5)

### 🔹 Script Created
    #!/bin/bash

    # Directories to backup
    dirs=("/home/kevin/docs" "/home/kevin/projects")

    # Backup location
    backup_dir="/home/kevin/backups"

    # Timestamp
    timestamp=$(date +"%Y-%m-%d_%H-%M-%S")

    # Create backup folder
    mkdir -p "$backup_dir/$timestamp"

    # Loop through directories
    for dir in "${dirs[@]}"
    do
        echo "Backing up $dir..."
        cp -r "$dir" "$backup_dir/$timestamp"
    done

    echo "Backup completed at $backup_dir/$timestamp"

---

### ⚡ Outcome
- Built real-world automation script  
- Used loops for multi-directory operations  
- Implemented timestamp-based backups  
- Simulated production-level DevOps workflow

## 🧠 Key Takeaways

- Loops enable **scalable automation**  
- `for` is best for known iterations  
- `while` is used for dynamic conditions
