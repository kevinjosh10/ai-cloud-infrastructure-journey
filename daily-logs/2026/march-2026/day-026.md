## 🔎 awk (Pattern Scanning & Data Processing)

### 🔹 Basic syntax
    awk '{action}' file

---

### 🔹 Example — Print first column
    awk '{print $1}' file.txt

---

### 🔹 Using delimiter (CSV)
    awk -F',' '{print $2}' data.csv

---

### 🔹 Example — Extract usernames
    awk -F':' '{print $1}' /etc/passwd

---

### ⚡ Outcome
- Learned column-based text processing  
- Parsed structured data efficiently

## ✂️ cut (Column Extraction Tool)

### 🔹 Basic syntax
    cut -d'delimiter' -f fields file

---

### 🔹 Example — Extract columns from CSV
    cut -d',' -f1,3 data.csv

---

### 🔹 Example — Extract system usernames
    cut -d':' -f1 /etc/passwd

---

### ⚡ Outcome
- Performed fast column extraction  
- Understood delimiter-based parsing  
- Compared cut vs awk usage

## 🔁 sed (Stream Editor for Text Transformation)

### 🔹 Replace text
    sed 's/old/new/g' file.txt

---

### 🔹 Modify file directly
    sed -i 's/error/warning/g' logfile.txt

---

### 🔹 Print specific lines
    sed -n '5,10p' file.txt

---

### ⚡ Outcome

## 🧪 Practice Script — User Creation Automation (Script 3/5)

### 🔹 Script Created
    #!/bin/bash

    INPUT="users.txt"
    LOG="user_creation.log"

    read -s -p "Enter default password: " DEFAULT_PASS
    echo ""

    while read username
    do
        [ -z "$username" ] && continue

        if id "$username" &>/dev/null; then
            echo "User $username already exists. Skipping..."
            echo "$(date) - Skipped existing user: $username" >> "$LOG"
        else
            sudo useradd "$username"
            echo "$username:$DEFAULT_PASS" | sudo chpasswd
            echo "User $username created."
            echo "$(date) - Created user: $username" >> "$LOG"
        fi

    done < "$INPUT"

    echo "Process completed!"

---

### ⚡ Outcome
- Automated user management  
- Implemented condition checks  
- Handled secure password input  
- Logged system operations
