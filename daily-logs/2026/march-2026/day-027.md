## 🧪 Practice Script — Disk Usage Reporter (Script 4/5)

### 🔹 Script Created
    #!/bin/bash

    THRESHOLD=80

    echo "Disk Usage Report:"
    echo "-------------------"

    df -h | grep -vE '^Filesystem|tmpfs|cdrom' | while read line; do
        usage=$(echo $line | awk '{print $5}' | sed 's/%//')
        partition=$(echo $line | awk '{print $1}')

        if [ $usage -ge $THRESHOLD ]; then
            echo "⚠️ Warning: $partition is ${usage}% full"
        else
            echo "✅ $partition is ${usage}% used"
        fi
    done

---

### ⚡ Outcome
- Monitored system disk usage  
- Parsed command output using awk and sed  
- Implemented conditional alerts  
- Built a real-world monitoring script
