### 2️⃣ Learned GitHub Actions Core Concepts

#### Workflow
- YAML automation files inside:
    .github/workflows/

#### Trigger (`on:`)
- Executes workflows automatically on events

Examples:
    on: push
    on: pull_request

#### Jobs
- Major workflow tasks like:
    lint
    test
    build
    deploy

#### Steps
- Individual commands/actions inside jobs

#### Runner
- Virtual machine executing workflow

Example:
    runs-on: ubuntu-latest

---

### 3️⃣ Created First GitHub Actions Workflow

Created:
    .github/workflows/lint.yml

Workflow performs:
- Repository checkout
- Python setup
- flake8 installation
- Python linting

Basic workflow structure learned:

```yaml
name: Python Linting

on:
  push:

jobs:
  lint:
    runs-on: ubuntu-latest
```

---

### 4️⃣ Learned Important Workflow Syntax

#### `uses:`
Used reusable GitHub Actions

Example:
    uses: actions/checkout@v4

Purpose:
- Downloads repository code into runner

---

#### `run:`
Executes shell commands

Examples:
    run: pip install flake8
    run: flake8 .

---

#### Python Setup Action
Used:
    actions/setup-python@v5

Configured:
    python-version: "3.11"
