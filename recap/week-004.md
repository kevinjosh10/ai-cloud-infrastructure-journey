## 🧠 What I Learned

### 🔹 Bash Scripting Fundamentals
- Shebang (`#!/bin/bash`)
- Variables and environment variables  
- Command substitution (`$(command)`)  
- User input using `read`  
- Arithmetic operations (`$(( ))`)  

---

### 🔹 Conditionals & Logic
- `if / elif / else` statements  
- Numeric comparisons (`-eq`, `-gt`, `-lt`)  
- String comparisons (`==`, `!=`, `-z`, `-n`)  
- File condition checks (`-f`, `-d`, `-r`, `-x`)  

---

### 🔹 Loops & Automation
- `for`, `while`, `until` loops  
- Iterating through files and directories  
- Using `break` and `continue`  

---

### 🔹 Functions & Script Structure
- Writing reusable functions  
- Passing arguments (`$1`, `$@`, `$#`)  
- Understanding script structure  
- Using exit codes (`exit 0`, `exit 1`)  

---

### 🔹 Debugging & Script Safety
- Debugging with `bash -x`  
- Using `set -x`, `set -e`, `set -u`  
- Printing variables for troubleshooting  

---

### 🔹 Text Processing
- `awk` for column-based extraction  
- `sed` for text replacement and filtering  
- `cut` for parsing structured data  

---

### 🔹 Editors & Productivity
- Using `nano` for quick edits  
- Using `vim` for efficient editing and navigation
