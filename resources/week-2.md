# 4. Terminal Environment

Recommended setup:

### Windows

Use **Windows Subsystem for Linux (WSL)**.

Install:

```
Ubuntu (WSL)
```

This provides a full Linux environment inside Windows.

---

### Mac / Linux

Use the built-in **Terminal**.

Example shells:

```
bash
zsh
```

# 5. Helpful Documentation

Linux manual pages can be accessed using:

```
man command
```

Example:

```
man ls
man grep
man find
```

This shows full documentation for commands.

# 7. Practice Suggestions

To reinforce learning this week:


- file creation

- permissions
- searching files
