## 🛠️ 4. Hands-On Tools (Essential)

### 🔧 Linux Networking Commands

Use these daily:

    ping google.com  
    traceroute google.com  
    nslookup google.com  
    dig google.com  
    ss -tulpn  
    netstat -tulpn  
    curl -v https://example.com  
    ip addr show  
    ipcalc 10.0.0.0/16  

---

### 📦 Install Required Tools (Ubuntu)

    sudo apt update  
    sudo apt install traceroute dnsutils net-tools ipcalc curl -y
