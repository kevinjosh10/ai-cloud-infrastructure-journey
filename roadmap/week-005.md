# 🗺️ Week 5 Roadmap – Networking Fundamentals I

**Focus:** IP Addressing, Subnetting, CIDR, DNS, DHCP, NAT  
**Goal:** Build a strong networking foundation required for cloud engineering  

---

## 🎯 Week Objective

By the end of Week 5, you will:
- Understand how IP addressing works  
- Perform subnetting quickly and accurately  
- Understand how DNS resolves domain names  
- Learn how devices get IP addresses using DHCP  
- Understand how NAT enables internet communication  
- Use networking tools to debug real-world issues  

---

## 📅 Day-by-Day Roadmap

---

## 📘 Day 29 – IPv4 Fundamentals

### 🔑 Topics
- IPv4 structure (32-bit, 4 octets)  
- Private IP ranges (RFC 1918)  
- Public vs Private IP  
- Loopback (127.0.0.1)  

### 🎯 Outcome
- Identify IP types (public/private)  
- Understand how devices are addressed in a network  

---

## 📘 Day 30 – Subnetting & CIDR

### 🔑 Topics
- CIDR notation (/24, /16, etc.)  
- Network vs Host bits  
- Subnet calculations  
- Using `ipcalc`  

### 🎯 Outcome
- Calculate network, broadcast, and host ranges  
- Solve subnetting problems confidently  

---

## 📘 Day 31 – IPv6 Basics

### 🔑 Topics
- IPv6 structure (128-bit, hexadecimal)  
- Abbreviation rules (::)  
- IPv6 types (global, link-local, loopback)  

### 🎯 Outcome
- Understand IPv6 addressing  
- Recognize importance in cloud environments  

---

## 📘 Day 32 – DNS (Domain Name System)

### 🔑 Topics
- DNS resolution flow  
- Record types (A, AAAA, CNAME, MX, TXT)  
- TTL (Time to Live)  
- Tools: `nslookup`, `dig`  

### 🎯 Outcome
- Perform DNS queries  
- Understand how domains resolve to IPs  

---

## 📘 Day 33 – DHCP & NAT

### 🔑 Topics
- DHCP (DORA process)  
- NAT (purpose and types)  
- Private → Public IP translation  

### 🎯 Outcome
- Understand automatic IP assignment  
- Understand how internal networks access the internet  

---

## 📘 Day 34 – Network Troubleshooting

### 🔑 Topics
- `ping` (connectivity testing)  
- `traceroute` (path tracing)  
- `ss` / `netstat` (ports and services)  
- `curl` (HTTP requests)  

### 🎯 Outcome
- Debug connectivity issues  
- Analyze network paths and services  

---

## 📘 Day 35 – Review & Practice

### 🔑 Tasks
- Subnetting drills (30 problems)  
- DNS practice using `dig`  
- Draw network diagram  
- Build networking cheat sheet  

### 🎯 Outcome
- Strong retention of all concepts  
- Ability to connect all networking components  

---
