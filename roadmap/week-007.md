## 🧭 Weekly Breakdown

### 📅 Day 43 – Setup & Variables
**Focus:** Environment setup + fundamentals  

#### Tasks
- Install Python 3.x  
- Set up virtual environment (`venv`)  
- Configure VS Code interpreter  
- Learn variables and data types  

#### Outcome
- Working Python environment  
- Basic script execution  

---

### 📅 Day 44 – Strings
**Focus:** Text processing (very important for real-world data)  

#### Tasks
- Learn string methods  
- Practice f-strings  
- Understand slicing  
- Work with multiline strings  

#### Practice
- Parse hostname:
  - Extract environment  
  - Extract role  
  - Extract domain  

#### Outcome
- Ability to process structured text data  

---

### 📅 Day 45 – Lists & Tuples
**Focus:** Managing collections of data  

#### Tasks
- Learn list operations  
- Practice list slicing  
- Use list comprehensions  
- Understand tuples  

#### Practice
- Create list of AWS regions  
- Filter only `us-` regions  

#### Outcome
- Efficient handling of multiple data points  

---

### 📅 Day 46 – Dictionaries & Sets
**Focus:** Structured data (core for cloud systems)  

#### Tasks
- Learn dictionaries  
- Practice dict operations  
- Use dict comprehensions  
- Learn sets  

#### Practice
- Model EC2 instance:
  - instance_id  
  - type  
  - state  
  - region  
  - tags  

#### Outcome
- Ability to represent real-world cloud objects  

---

### 📅 Day 47 – Loops & Conditionals
**Focus:** Automation logic  

#### Tasks
- Learn for loops  
- Learn while loops  
- Practice conditionals  
- Use control statements  

#### Practice
- Filter EC2 instances:
  - Only running  
  - Only specific region  

#### Outcome
- Decision-making and automation logic  

---

### 📅 Day 48 – Functions & Modules
**Focus:** Reusable and modular code  

#### Tasks
- Define functions  
- Use return values  
- Learn *args and **kwargs  
- Understand modules  

#### Practice
- Create helper module:
  - Format instance ID  
  - Extract region  
  - Calculate runtime  

#### Outcome
- Clean and reusable code structure  

---

### 📅 Day 49 – File I/O & JSON
**Focus:** Real-world data handling  

#### Tasks
- Read files using `with`  
- Write and append files  
- Learn JSON parsing  
- Work with JSON files  

#### Practice
- Create AWS-like JSON data  
- Filter instances by state  
- Save filtered results  

#### Outcome
- Built a data pipeline:
  
    READ → PROCESS → FILTER → SAVE
