## 🔄 Execution Strategy

Follow this daily pattern:

1. Learn concept (30–40 mins)  
2. Implement code (60–90 mins)  
3. Break things intentionally  
4. Debug and fix  
5. Refactor code  

---

## ⚡ Key Priorities

- Focus on understanding, not memorization  
- Always test your code  
- Keep scripts modular and clean  
- Avoid hardcoding values  
- Use real-world scenarios  

---

## 🚧 Common Mistakes to Avoid

- Not handling exceptions properly  
- Ignoring API failures or timeouts  
- Hardcoding AWS regions or values  
- Writing unstructured, messy code  
- Skipping testing
